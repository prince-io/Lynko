# components/icons — SVG Icon Components

## Ownership

Owns all reusable SVG icon components used across the app.

## Local Contracts

- Each component accepts `{ w, h, className }` props (defaults: `w=15`, `h=15`, `className=""`)
- Components render inline SVG elements with `width`, `height`, `viewBox`, and `className` set from props
- All components are default exports, re-exported from `index.js`
